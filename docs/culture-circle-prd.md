## 2. 创意介绍

### 2.1 创意名称

**同频圈** — 传统文化艺术兴趣圈子匹配平台

### 2.2 想解决什么问题

随着"国潮"复兴和全民健康意识的提升，越来越多人想学习太极拳、中医养生功法、民乐、书法等中华传统文化艺术，但在城市中苦于找不到附近的学习圈子或志同道合的伙伴。线下课程信息分散、社群难以触达、兴趣匹配效率极低，导致大量爱好者"想学无处去、想练无人陪"。

### 2.3 为什么会想到做这个

灵感来源于身边真实的痛点——许多朋友想练太极拳强身健体，却不知道小区附近就有晨练队伍；有人想学古筝，却找不到同城的学习伙伴。传统文化艺术的传承，人与人的连接是关键。我们希望通过技术手段，让"同频"的人在城市中轻松相遇。

### 2.4 产品形态

**同频圈**是一款基于地理位置的社交匹配小程序（微信小程序 + 移动端 App），用户发布自己的定位和感兴趣的文化艺术类别，系统自动在 1km、5km、10km、30km 范围内匹配同频的人，并通过微信或手机号建立联系。

## 3. 目标用户及痛点

### 3.1 面向哪些用户

| 用户群体 | 描述 |
|----------|------|
| 传统文化爱好者 | 对太极拳、书法、民乐、茶道等传统文化艺术感兴趣，希望找到学习伙伴或加入本地圈子的人 |
| 养生健康人群 | 关注身心健康，想通过中医养生功法、太极、八段锦等方式锻炼身体的中青年及中老年群体 |
| 文化传承者 | 传统文化艺术的从业者或资深爱好者，希望扩大影响力、招收学员、组织线下活动，可发布兴趣圈子 |
| 新城市居民 | 刚搬到新城市或新社区，想通过共同的文化兴趣快速融入当地社交圈的人 |

### 3.2 使用场景

- 下班后想找附近太极拳晨练队加入
- 周末想约同城书法爱好者交流
- 搬到新社区想快速找到民乐练习伙伴
- 退休后想系统学习中医养生功法

打开**同频圈**，一键发布兴趣标签，即刻发现身边同频之人。

### 3.3 当前痛点

| 痛点 | 现状 | 同频圈的解决方案 |
|------|------|------------------|
| 信息不对称 | 公园里的太极队、社区书法班信息靠口口相传，新人完全不知道 | LBS 定位自动发现附近所有相关圈子 |
| 匹配效率低 | 在微信群、贴吧、小红书大海捞针，难以筛选距离和兴趣 | 按距离（1/5/10/30km）+ 兴趣标签精准匹配 |
| 联系门槛高 | 找到信息后不知道如何联系，缺乏信任基础 | 平台内微信/手机号直联，降低社交门槛 |
| 持续性差 | 一个人练容易放弃，缺乏同伴激励 | 匹配同频伙伴，形成稳定的练习社交圈 |

## 4. 核心功能设计

### 4.1 用户使用流程

平台支持两类角色的使用流程：**兴趣爱好者**寻找同频的人和圈子，**老师/传承人**创建并发布兴趣圈子。

#### 4.1.1 兴趣爱好者流程

```
注册登录 → 选择兴趣 → 发布定位 → 智能匹配（同频的人 + 同频的圈子）→ 建立联系
```

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1. 注册登录 | 微信手机快捷号登录或手机短信验证码登录，完善个人资料 | 降低注册门槛，双方式覆盖微信与非微信环境 |
| 2. 选择兴趣 | 勾选感兴趣的文化艺术类别 | 支持多选 |
| 3. 发布定位 | 授权地理位置，输入关键词搜索具体兴趣，一键发布 | 支持手动调整位置，兴趣关键词匹配搜索 |
| 4. 智能匹配 | 自动发现附近同频的人与同频的圈子 | 按距离分级展示，人 + 圈子双维度匹配 |
| 5. 建立联系 | 微信/手机号直接沟通 | 隐私保护机制 |

#### 4.1.2 老师/传承人流程

```
注册登录 → 选择兴趣 → 发布定位 → 创建圈子 → 圈子被匹配 → 学员联系
```

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1. 注册登录 | 微信手机快捷号登录或手机短信验证码登录，完善个人资料 | 降低注册门槛，双方式覆盖微信与非微信环境 |
| 2. 选择兴趣 | 勾选感兴趣的文化艺术类别 | 支持多选 |
| 3. 发布定位 | 授权地理位置，输入关键词搜索具体兴趣，一键发布 | 支持手动调整位置，兴趣关键词匹配搜索 |
| 4. 创建圈子 | 填写圈子信息（标题、兴趣标签、描述、位置、联系方式） | 详见 4.4 圈子发布功能 |
| 5. 圈子被匹配 | 系统自动将圈子推荐给附近兴趣爱好者 | 按距离分级展示 |
| 6. 学员联系 | 兴趣爱好者通过微信/手机号联系老师 | 隐私保护机制 |

### 4.2 兴趣关键词搜索功能

#### 4.2.1 功能描述
用户在选择兴趣时，支持输入关键词搜索具体的传统文化艺术项目，系统通过关键词匹配返回相关兴趣标签，帮助用户精准表达需求。

#### 4.2.2 交互流程
1. 用户进入"选择兴趣"页面
2. 页面展示预设的兴趣大类（武术养生、民族器乐、书画篆刻等）
3. 用户可在顶部搜索框输入兴趣关键词
4. 系统实时返回匹配的兴趣标签列表
5. 用户点击选择目标标签，或选择"自定义添加"
6. 确认后该标签加入个人兴趣档案

#### 4.2.3 关键词匹配规则
- **支持具体项目名称匹配**：如"陈氏太极拳养生八式"、"杨氏太极拳85式"、"八段锦国家体育总局版"
- **支持流派/风格匹配**：如"颜体楷书"、"王派快板"、"梅派京剧"
- **支持作品/曲目匹配**：如"兰亭序临摹"、"梅花三弄"、"高山流水"
- **支持技法/功法匹配**：如"逆锋起笔"、"云手"、"吐纳法"
- **支持拼音首字母搜索**：如"cstj"匹配"陈氏太极拳"
- **支持模糊匹配**：输入"太极"可匹配所有太极拳相关项目
- **支持自定义标签**：若系统中无匹配项，用户可自定义输入并创建新标签

#### 4.2.4 兴趣标签体系

系统内置三级标签体系：

| 一级分类 | 二级分类示例 | 三级具体项目示例 |
|----------|--------------|------------------|
| 武术养生 | 太极拳 | 陈氏太极拳养生八式、杨氏太极拳85式、吴氏太极拳、孙氏太极拳 |
| 武术养生 | 气功功法 | 八段锦（国家体育总局版）、五禽戏、易筋经、六字诀 |
| 民族器乐 | 弹拨乐器 | 古筝（渔舟唱晚、高山流水）、琵琶（十面埋伏、阳春白雪） |
| 民族器乐 | 拉弦乐器 | 二胡（二泉映月、赛马）、京胡、马头琴 |
| 书画篆刻 | 书法 | 颜体楷书临摹、兰亭序行书、曹全碑隶书、峄山碑篆书 |
| 书画篆刻 | 国画 | 工笔花鸟、写意山水、水墨人物、青绿山水 |
| 茶道花艺 | 茶艺 | 工夫茶冲泡、宋代点茶、普洱茶品鉴、白茶冲泡 |
| 戏曲曲艺 | 京剧 | 梅派唱腔、程派水袖、老生唱腔、花脸脸谱绘制 |

#### 4.2.5 技术实现
- **分词检索**：基于 Jieba/THULAC 中文分词，支持语义理解
- **标签库管理**：维护传统文化艺术项目知识库，支持动态扩展
- **用户自定义标签审核**：新标签经简单审核后纳入系统词库
- **搜索联想**：输入时实时展示热门相关标签
- **搜索历史**：记录用户常用搜索词，快速复选
- **标签权重计算**：用户选择的标签细化程度越高，匹配精度越高

### 4.3 匹配机制

基于地理位置的同心圆匹配机制，支持**同频的人**和**同频的圈子**双维度匹配：

- **1km 范围**：步行可达，适合日常练习组队
- **5km 范围**：短途骑行/驾车，适合周末活动
- **10km 范围**：同城区域，适合大型聚会
- **30km 范围**：城市周边，适合特色活动/名师寻访

#### 4.3.1 同频的人匹配

用户发布定位和兴趣后，系统在指定范围内寻找兴趣相同的人。

匹配排序规则：距离权重（40%）+ 兴趣重合度（40%）+ 活跃度（20%）

#### 4.3.2 同频的圈子匹配

用户发布定位和兴趣后，系统同时在指定范围内寻找兴趣匹配的圈子。圈子由老师/传承人创建并发布，爱好者可查看圈子详情并主动联系。

圈子匹配排序规则：距离权重（30%）+ 兴趣标签重合度（50%）+ 圈子活跃度（20%）

匹配结果中，**同频的人**和**同频的圈子**分两个列表展示，用户可切换查看。

### 4.4 圈子发布功能

#### 4.4.1 功能描述

传统文化艺术的老师或传承人可以创建并发布兴趣圈子，供附近的兴趣爱好者搜索和加入。圈子是平台连接老师与学员的核心载体，让有组织的学习社群更容易被发现。

#### 4.4.2 谁可以发布圈子

- 传统文化艺术的从业者（如太极拳老师、书法老师、民乐教师）
- 资深爱好者或传承人，希望组织线下练习、教学或交流活动
- 文化机构/工作室，希望扩大影响力、招收学员

#### 4.4.3 圈子信息字段

| 字段 | 说明 | 是否必填 |
|------|------|----------|
| 标题 | 圈子名称，如"朝阳公园陈氏太极拳晨练班" | 必填 |
| 兴趣标签列表 | 关联一个或多个兴趣标签，如"陈氏太极拳"、"养生八式" | 必填 |
| 圈子描述 | 圈子的详细介绍，包括教学内容、练习安排、适合人群等 | 必填 |
| 位置信息 | 圈子活动地点，支持地图选点 | 必填 |
| 联系电话 | 老师的联系电话 | 选填 |
| 微信 | 老师的微信号 | 选填 |
| 活动时间 | 圈子的活动时间安排，如"每周六早 7:00" | 选填 |
| 人数上限 | 圈子容纳人数限制 | 选填 |

#### 4.4.4 发布交互流程

1. 老师/传承人点击"创建圈子"进入发布页面
2. 填写圈子标题
3. 通过兴趣关键词搜索选择关联的兴趣标签（支持多选）
4. 填写圈子描述信息
5. 在地图上选择活动位置，支持搜索具体地点
6. 填写联系电话和/或微信号
7. 可选：填写活动时间、人数上限
8. 确认发布，圈子进入系统并等待匹配

#### 4.4.5 圈子展示与联系

- 兴趣爱好者在匹配结果中看到同频圈子列表
- 点击圈子卡片查看完整详情（标题、描述、标签、位置、活动时间等）
- 联系方式（电话/微信）在用户主动点击"联系老师"时展示，保护隐私
- 圈子创建者可管理圈子信息、查看被联系次数

## 5. 价值与意义

### 5.1 社会价值

**促进传统文化活态传承**：让传统文化艺术从"博物馆里的展品"回归日常生活。通过连接同频之人，形成自发的学习社群和练习圈子，让太极拳、书法、民乐等在社区中自然传播，真正实现"文化自信"从口号到行动。

### 5.2 健康价值

**推动全民身心健康**：太极拳、八段锦、中医养生功法等已被科学证实对改善身心健康有显著效果。通过降低参与门槛（就近找到伙伴），帮助更多人养成规律练习习惯，缓解现代城市人的焦虑和亚健康状态。

### 5.3 文化价值

**构建新型文化社交网络**：以传统文化艺术为纽带，打破城市中人与人之间的疏离感。让有共同文化兴趣的人跨越年龄、职业的界限，建立真实、有温度的线下社交关系，增强社区凝聚力和文化认同感。
